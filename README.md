## Setup

```
bin/setup
```

* start postgresql if it not active below
```
sudo service postgresql start
```
* migrate database
```
bundle exec rails db:migrate
```
* create database seed
```
bundle exec rails db:seed
```
## start:

```
bundle exec rails s
```
* second panel with
```
bin/webpacker-dev-server
```
